# 🐟 大鱼吃小鱼 (Big Fish Eat Small Fish)

一个用 HTML5 Canvas 制作的大鱼吃小鱼网页游戏。

## 🎮 玩法

- **移动鼠标**控制大鱼游动
- **吃掉**比你小的鱼来成长变大
- **避开**比你大的鱼，否则会被吃掉！
- 游戏会越来越刺激，AI 鱼之间也会互相捕食

## 🚀 运行方式

直接在浏览器中打开 `index.html` 即可开始游戏。

或者用任意 HTTP 服务器：

```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .
```

然后访问 `http://localhost:8000`

## ✨ 特性

- 🎨 精美的手绘风格鱼类图形
- 🌊 动态水下场景（光线、气泡、海草）
- 🐠 多种颜色和花纹的鱼类
- 🧠 AI 鱼类自主行为（追猎、逃跑、游荡）
- 📱 支持触屏设备
